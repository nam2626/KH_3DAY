## 1. 로그인 페이지(형식상)

- 실제 로그인 수행 X

## 2. 메인 페이지

-페이지 로드시
회원 정보 전체 조회

- 기능
  회원정보 수정 버튼, 회원정보 삭제 버튼
  삭제 버튼 클릭하면 회원정보 삭제 기능 수행 데이터를 리로드

## 3. 회원 정보 수정 페이지

- 페이지 로드시
  해당 회원 정보 조회 해서 폼에 데이터를 바운드
- 기능
  수정버튼 클릭하면 수정 후 메인 페이지로 이동
  취소버튼 클릭하면 메인페이지로 이동

## 4. 회원정보 추가 페이지

- 입력받을 회원 정보 폼을 작성
- 등록버튼 클릭시 입력 회원 정보를 유효성 검사 후 등록처리
- 취소버튼 클릭시 다시 메인페이지로 이동
